# X-PAND
X-PAND is a four-year international multi-centric project funded by the European Union under the Horizon Europe – European Innovation Council program. The ambitious goal of the project is to exploit innovative hematopoietic stem cell manipulation protocols in combination with a deep multiomics profiling to bring novel, efficient and safer hematopoietic stem cell gene therapies to clinical application.

In this repository, you can find the standardized pipelines used within the project for the single-cell omics analysis. 
You can also find the specific scripts used within each specific task.
